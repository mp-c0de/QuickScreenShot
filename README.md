# QuickScreenShot

A lightweight macOS utility for quickly capturing screen regions with a single keypress.

## Features

- Select a screen region once, capture it repeatedly
- Configurable global hotkey (default: press `0`) â€” change it in Settings
- Screenshots saved with auto-incrementing numbers (1.png, 2.png, etc.)
- Three quality levels: Low, Medium, Best (Retina)
- Settings window (Cmd+,) with General, How to Use, and About tabs
- Remembers all settings between launches (region, save location, shortcut)
- Mac App Store ready with sandbox and security-scoped bookmarks

## Usage

1. Click **Set Coordinates** and drag to select a region
2. Click **Set Location** to choose a save folder
3. Press your shortcut key or click **Capture** to take screenshots
4. Open **Settings** (Cmd+, or gear icon) to change the shortcut

## Requirements

- macOS 14.0+
- Screen Recording permission
- Accessibility permission (for global hotkey)

## Build

Open `QuickScreenShot.xcodeproj` in Xcode and run.
